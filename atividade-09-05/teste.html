O que é HTML?

HTML, ou HyperText Markup Language, é uma linguagem de marcação de texto comumente utilidada para a programação na internet e online. Todo documento HTML possui as tags, que são palavras marcadoras que têm a função ditam a formatação da linguagem de marcação.
